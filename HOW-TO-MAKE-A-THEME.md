# How To Make A Theme
## From Scratch:
1. make a new folder
2. make a index.html
3. fill the index.html with your theming styles (use window.cink.unlock(); to mark your lock screen as completed)
4. put it in a zip (ensure index.html is at the root of your zip, use commands like Compress-Archive on windows, and zip on linux)
## From Template:
1. run `git clone https://www.github.com/Fundiman/cink-theming` in the terminal of your choice in any directory
2. run `cd cink-theming` to change directory to that folder
3. read the EDITING.md file for futher instructions on how to edit
